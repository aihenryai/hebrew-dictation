# Samples Directory

כל תת-תיקייה = sample יחיד. הסקריפט סורק את התיקיות פה בסדר אלפביתי.

## מבנה

```
samples/
  sample_01_quiet/
    audio.wav           # חובה — WAV מועדף, גם MP3/M4A/OGG עובד
    reference.txt       # חובה — תמלול ייחוס בעברית (UTF-8)
```

## דרישות אודיו

- **פורמט:** WAV (מועדף), MP3, M4A, OGG
- **משך:** 15-30 שניות (שומר על זמן רצה סביר)
- **ערוצים:** Mono מועדף
- **Sample rate:** 16kHz הוא native ל-Whisper, אבל גבוה יותר גם יעבוד

## פורמט reference

טקסט רגיל UTF-8. השתמש בניקוד רגיל (פסיקים, נקודות). **אל תכלול** timestamps או תגי דובר.

דוגמה ל-`reference.txt`:
```
שלום, היום אני רוצה לדבר איתכם על כלי בינה מלאכותית חדש שיצא השבוע. הוא נקרא Claude ואני חושב שזה הדבר הכי מעניין שראיתי לאחרונה.
```

## המלצות הקלטה

- דבר בקצב טבעי (אל תאט לטובת המיקרופון)
- **גיוון קריטי:** חדר שקט, רעש קל, קצב מהיר, מונחים טכניים
- שם הקובץ בפורמט `sample_NN_description` כדי שהדוח יהיה קריא
- כדאי להקליט **לפחות sample אחד ישירות דרך האפליקציה** (real-world)

## אל תעלה ל-git

`samples/*/` מוגדר ב-`.gitignore` — רק קובץ ה-README הזה יישמר. זה נועד להגן על פרטיות הקלטות.
